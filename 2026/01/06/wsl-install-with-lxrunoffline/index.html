用 LxRunOffline 安装 WSL

1. 安装 LxRunOffline

从软件 Github 仓库的 Release 页面 下载程序。目前最新版本为v3.5.0，其 mingw 版本 与 msvc 版本 均可。

2. 下载 WSL 安装包

在微软提供的 安装指南 中可以看到不同版本的下载链接。以 Ubuntu 22.04 为例，下载完成后得到 Ubuntu2204-221101.AppxBundle 文件，将其解压缩（可直接用 7-zip 解压缩，或者改后缀名为 .zip 后解压缩），得到的文件夹内包含了多个后缀为 .appx 的文件。

根据设备的架构选择 Ubuntu_2204.1.7.0_x64.appx 或 Ubuntu_2204.1.7.0_ARM64.appx，将其解压缩，得到的文件夹内的目录结构如下：

3. 用 LxRunOffline 安装 WSL

在开始安装前，按照 此微软安装文档 的要求，启用相应的可选功能。

在终端中打开 LxRunOffline 解压缩后的目录。为确保兼容性，LxRunOffline 程序目录与 WSL 安装包目录的路径建议不要有中文。
